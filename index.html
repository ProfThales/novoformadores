Projeto JEPP Colégio Sirley Jagas

Projeto de Empreendedorismo

Colégio Estadual Sirley Jagas

O nosso projeto de empreendedorismo JEPP (Jovens Empreendedores Primeiros Passos) é uma parceria entre o Colégio Estadual Sirley Jagas e o SEBRAE.

O projeto visa incentivar o espírito empreendedor e a orientação para negócios nas novas gerações de forma lúdica e prática.
